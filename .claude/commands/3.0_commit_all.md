Commit alle endringer til lokalt repo.

1. Kjør `git status` for å se utrackede og endrede filer.
2. Kjør `git diff` for å se endringer.
3. Kjør `git log --oneline -3` for å se siste commit-stil.
4. Stage alle relevante filer (ikke .env eller secrets).
5. Skriv en kort, beskrivende commit-melding på norsk eller engelsk basert på endringene.
6. Commit lokalt. IKKE push. IKKE endre andre branches.
