Du skal oversette akademisk norsk tekst fra `rapport_1/` til akademisk engelsk i `rapport_2/`. Oppgavens tittel er "Beyond Manual Review: How RAG-Powered Agents Transform Norwegian Real Estate Document Analysis".

Basert på brukerens input `$ARGUMENTS`, finn riktig kildefil i:

## Steg 2: Les kildefilen

Les den norske kildefilen komplett. Forstaa innholdet, strukturen, referansene og figurreferansene.

## Steg 3: Oversett til akademisk engelsk

### Overordnede regler

1. **Akademisk engelsk** med formelt register (tilsvarende tonen i den norske teksten)
2. **Behold ALLE referanser uendret**: (Singh et al., 2025, R7) forblir identisk
3. **Behold ALLE figurreferanser uendret**: [VIS Fig 4: ...], Fig 23, Tabell 3.1 etc.
4. **Behold ALLE tall og metrikker uendret**: 73.4%, 29.4%, 12.1 prosentpoeng → 12.1 percentage points
5. **Behold seksjonsnummerering**: ## 2.1, ## 5.4.2 etc.
6. **Behold markdown-formatering**: **bold**, *italic*, overskriftsnivaaer

### Spraaklige regler

- **Fagtermer som IKKE oversettes** (behold paa engelsk/uendret): RAG, agentic, embedding, chunking, retrieval, prompt, pipeline, agent, precision, recall, F1-score, planning, reflection, tool use, Ground Truth, false positive, false negative, chunks, tokens, ChromaDB, Claude, Voyage AI, cosine similarity, top_k, temperature
- **Norske fagtermer som oversettes**:
  - salgsoppgave → sales prospectus / property listing
  - tilstandsrapport → condition report
  - kjoepekontrakt → purchase contract
  - eiendomstransaksjon → real estate transaction
  - boligkjoeper → home buyer
  - megler → real estate agent
  - takstmann → property surveyor / appraiser
  - avhendingsloven → the Norwegian Sale of Real Property Act (Avhendingslova)
  - TG-gradering → TG rating (condition grade)
  - budrunde → bidding round
  - prosentpoeng → percentage points
  - risikokategori → risk category
  - dokumentsett → document set
  - ordbudsjett → word budget
- **"vi" → "we"** (akademisk we)
- **Norske stedsnavn og lovnavn**: Behold paa norsk med engelsk forklaring ved foerste forekomst, f.eks. "the Avhendingslova (Norwegian Sale of Real Property Act)"
- **finn.no**: Behold som er (det er et egennavn)

### Stilregler (fra skrivestilkromosomet, tilpasset engelsk)

- **Tone**: Academic substance with accessible tone, never dry, never informal
- **Sentence rhythm**: ~20 words average, vary deliberately — long explanatory → short punch sentence
- **Hedging**: Where uncertain ("suggests", "may indicate"), directness where results are clear
- **Active voice** preferred: "we implemented" not "it was implemented"
- **Sentences under 30 words**. Break up long sentences with periods.

### Unngaa (samme regler som norsk, tilpasset engelsk)

**Tegnsetting:**
- **No em dash** — use comma, period, colon, or parentheses instead
- **No semicolon overuse** — use short sentences instead

**Words and expressions that must NEVER be used:**
- Generic filler: ~~"valuable learning"~~, ~~"interesting findings"~~, ~~"it is worth noting"~~
- Sales words: ~~"innovative"~~, ~~"groundbreaking"~~, ~~"seamless"~~, ~~"robust"~~, ~~"sophisticated"~~
- AI markers: ~~"it is important to note"~~, ~~"in this context"~~, ~~"it should be mentioned"~~, ~~"this enables"~~, ~~"this underscores"~~
- Excessive adverbs: ~~"significantly"~~ (use "substantially" or concrete numbers), ~~"fundamentally"~~, ~~"remarkably"~~
- Empty transitions: ~~"Furthermore"~~, ~~"Moreover"~~, ~~"In addition to this"~~, ~~"It is also worth noting"~~

### Sannhetsregler

- **Oversett kun det som staar i kilden**. Ikke legg til ny informasjon, nye referanser, eller nye tolkninger.
- **Ikke endre tall**, metrikker, eller paastander.
- **Ikke fjern innhold**. Alt i kildefilen skal med i oversettelsen.
