Merge nåværende branch inn i main.

1. Kjør først kommandoen fra C:\Kodeprosjekter\master\.claude\commands\3.0_commit_all.md (commit alle endringer).
2. Noter nåværende branch-navn.
3. Bytt til main: `git checkout main`.
4. Merge feature-branchen inn i main: `git merge <branch-navn>`.
5. Bytt tilbake til den opprinnelige branchen.
6. IKKE push. IKKE slett branches.
