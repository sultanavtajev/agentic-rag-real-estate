Avslutt alle kjørende prosjektprosesser (python, node, uvicorn) og slett __pycache__.

1. Sjekk om det er prosesser som lytter på port 8000 (backend) og 3000 (frontend).
2. Sjekk om det finnes kjørende python- og node-prosesser.
3. Drep alle python- og node-prosesser med: `powershell -Command "Get-Process python,node -ErrorAction SilentlyContinue | Stop-Process -Force"`
4. Slett alle __pycache__-mapper i prosjektet med: `find /c/Kodeprosjekter/master -type d -name "__pycache__" -exec rm -rf {} + 2>/dev/null`
5. Verifiser at portene 8000 og 3000 er ledige.
6. Rapporter hva som ble avsluttet og ryddet.
